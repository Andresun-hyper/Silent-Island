# 孤岛疗愈 App PRD

## 产品定位

`Ink Poles` 是一个孤疗愈风的互动墨水场景 app。它不是传统游戏，也不是纯动画展示，而是一个安静、可触碰、可停留的情绪空间。

首版大方向：把单一动态疗愈场景做完整、克制、有质感，先不扩展成复杂多页面产品。

## 核心体验

- 打开页面后，浅灰纸面中逐层出现电线杆、小路、草地、电线和手写造景。
- 用户可拉扯电线；拉得越紧，线的抖动越弱，释放后缓慢回弹。
- 停留或交互会触发手写笔触聚合成鸟、月或孤岛造景。
- 造景会停留、呼吸、再在重置时散成字迹。

## 首版功能范围

- 保留现有静态镜头、line boiling、catenary sway、胶片噪点和构图位置。
- 继续使用棕色、赭石、焦褐、低透明灰橄榄草地，不做大面积绿色草坪。
- 小路两侧保留明显留白。
- 增加手写字体造景：鸟、月、孤岛。
- 增加入场、造景聚合、静默停留、出场散开、重置的完整状态闭环。
- 增加极简控制层：造景切换、重置、会话时间和拉线次数。

## 验收标准

- 页面 3 秒内出现完整主场景入场。
- 电线可鼠标或触摸拖拽，张力越强抖动越弱。
- 至少一个手写造景具备入场、停留、出场动画。
- UI 不遮挡主画面，不出现黑色开发角标或强暗角。
- 桌面和移动端都能正常展示，Canvas 不空白。
