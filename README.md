# Ink Poles

A standalone Next.js version of the ink-wash utility pole animation.

The visual effect is isolated from the Eazo platform shell. It only uses:

- `src/app/page.tsx`
- `src/components/ink-poles-wrapper.tsx`
- `src/components/ink-poles-canvas.tsx`
- `src/lib/animation/*`

## Getting Started

Install dependencies:

```bash
npm install
```

Start the local preview:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Build a static export:

```bash
npm run build
```

The generated site is written to `out/`.
